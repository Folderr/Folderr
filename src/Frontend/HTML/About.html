About Evolve-X

Evolve-X is an image host and link shortener. Evolve-X is meant to be lightweight.
While being lightweight and fast can not always be guaranteed with many security features, we use a very well structured backend to power everything
Evolve-X is made purely in JS, HTML, and CSS. No php or python here.

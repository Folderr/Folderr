About

I wanted to make something that had an easy interface and was easy to set up, with the benefit of being lightweight and fast. I found that most applications that did this relied on using Python. Evolve X will use a strict API to
improve efficiency and prevent breakage.

Evolve X is still in its early stages, and documentation is being written for it but is not complete

Evolve-X is a TypeScript based NodeJS project made by Null, Decease, and Catbirby.<br>You can download Evolve-X via Git at our GitLab repository

Why Evolve-X?

Lightweight

We designed Evolve-X to use as few resources as possible while being secure and speedy

Self-hostable

Evolve-X is not meant to be only hosted by us, you can host it too.

Open Source

Evolve-X is an open source NodeJS TypeScript project to allow you to easily change the code

Secure

Evolve-X was made with your security in mind. We track as little as possible.
